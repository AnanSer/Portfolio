# Command Palette - Quick Start Guide

## How to Use

### Open the Command Palette

**Method 1: Keyboard Shortcut** (Recommended)
- **Windows/Linux**: Press `Ctrl + K`
- **Mac**: Press `⌘ + K`

**Method 2: Floating Button**
- Click the gradient search button in the bottom-right corner

---

## What You Can Do

### 1. Navigate Sections
Type any of these to jump to sections:
- `home` - Go to Home section
- `about` - Go to About section
- `journey` - Go to Journey timeline
- `workspace` or `dashboard` - Go to Developer Workspace
- `services` - Go to Services section
- `projects` - Go to Projects gallery
- `testimonials` - Go to Testimonials
- `contact` - Go to Contact form

### 2. Quick Actions
- `theme` - Toggle between light and dark mode
- `resume` or `download` - Download resume
- `github` - Open GitHub profile
- `linkedin` - Open LinkedIn profile
- `ai` or `assistant` - Open AI Assistant

### 3. Search Projects
Start typing any project name or technology:
- `traffic` - Traffic Fine Management System
- `letter` - Letter Management System
- `expense` or `budget` - Expense Tracker
- `food` - Food Order App or Food Website
- `task` - Task Management
- `next` - Shows all Next.js projects
- `react` - Shows all React projects
- `mongodb` - Shows all MongoDB projects

---

## Keyboard Navigation

Once the palette is open:

- **↑** / **↓** - Move up/down through results
- **Tab** - Cycle to next result
- **Enter** - Execute selected command
- **Escape** - Close the palette

---

## Pro Tips

1. **Empty Search**: With no search query, see all available commands

2. **Quick Access**: The palette appears instantly - no loading delay

3. **Fuzzy Search**: Type partial words:
   - `traf` finds "Traffic Fine Management"
   - `serv` finds "Services"
   - `about` finds "About"

4. **Technology Search**: Search by tech stack:
   - `node` shows all Node.js projects
   - `firebase` shows Firebase projects
   - `tailwind` shows Tailwind CSS projects

5. **Multiple Results**: Use arrow keys to navigate when multiple items match

6. **Auto-Close**: Palette closes automatically after executing a command

## Troubleshooting

**Q: Ctrl+K doesn't work**
A: Make sure no input field has focus. If you're typing in a form, click outside first.

**Q: Palette appears empty**
A: Try refreshing the page. The projects array should load automatically.

**Q: Can't close the palette**
A: Press Escape or click the dark area outside the palette window.

**Q: Keyboard navigation not working**
A: Make sure the palette has focus (it should auto-focus when opened).

**Q: Floating button is blocking content**
A: The button is positioned in the bottom-right. You can scroll the page normally; it won't interfere.
